A redesign project for DoktorABC.

## Overview

This project contains the redesigned implementation of DoktorABC.

## Getting Started

### Prerequisites

- Node.js (version 18+)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Testing

```bash
npm test
```
