# Exports

This directory contains generated HTML reports, CSV data exports, and other exported artifacts from the redesign project.

## Contents

- `excel-migration-guide.html` - HTML guide for Excel migration processes
- `project-status-onepager.html` - One-page project status summary (HTML format)
- `redesign-analytics-review-table.csv` - Analytics review table data export

## Usage

These files are typically generated from documentation sources and contain:
- Formatted HTML reports for easy viewing
- CSV data for spreadsheet analysis
- Export artifacts for stakeholder distribution

Files in this directory may be regenerated from source documentation as needed.
