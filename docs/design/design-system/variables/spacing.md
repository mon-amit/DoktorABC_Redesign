# Design System Spacing

This document contains all spacing variables extracted from the Figma design system.

## Gap Spacing

| Token | Value | Usage |
|-------|-------|-------|
| `gaps/0` | 0 | No gap |
| `gaps/8` | 8 | Small gap |
| `gaps/12` | 12 | Medium-small gap |
| `gaps/16` | 16 | Medium gap |
| `gaps/24` | 24 | Large gap |
| `gaps/32` | 32 | Extra large gap |
| `gaps/60` | 60 | Maximum gap |

## Vertical Gap Spacing

| Token | Value | Usage |
|-------|-------|-------|
| `vertical_gaps/xxl` | 110 | Extra extra large vertical gap |

## Button Spacing

| Token | Value | Usage |
|-------|-------|-------|
| `button/gap` | 8 | Gap between button elements |
| `button/padding_h` | 32 | Horizontal button padding |
| `button/padding_h_icon` | 16 | Horizontal padding for icon buttons |
| `button/padding_v` | 20 | Vertical button padding |
| `button/padding_v_icon` | 16 | Vertical padding for icon buttons |
