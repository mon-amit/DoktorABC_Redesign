## Usage Guidelines

### Font Family Hierarchy

1. **Poppins**: Headers, titles, and display text
2. **Roboto**: Body text and content

### Font Weight Scale

- `400` (Regular): Body text, normal emphasis
- `500` (Medium): Subtle emphasis, labels
- `600` (SemiBold): Headers, important labels
- `700` (Bold): Strong emphasis, mobile titles

### Responsive Typography

- Display sizes: Large hero text, marketing pages
- Header sizes: Section headers, page titles
- Subtitle sizes: Sub-headers, card titles
- Text sizes: Body content, descriptions
- Label sizes: Form labels, buttons, UI elements
