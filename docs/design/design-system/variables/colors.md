## Usage Notes

- All colors are defined as hex values
- Primary colors should be used for main brand elements
- Secondary colors for supporting elements
- Text colors follow a hierarchy: primary > secondary > tertiary
- Surface colors define background contexts
- Border colors provide visual separation
- Icon colors adapt to their context
